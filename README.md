# Assistive-Communication-Device---Team160-The-Goldstein-Dolphins-MIT-Cre-at-e-Challenge


This is a project made for the MIT Cre[at]e Challenge. This was for the 2026 season. 

Team [160] The Goldstein Dolphins created an assistive device that allows our co-designer DJ, a 6th grader on the autism spectrum who has trouble with nonverbal cues like rasing his hand, to more effectively communicate with his teachers and paraprofessionals through a button-actuated communication protocole.
